# Automotive 设计风格 Skills

## 覆盖范围
本 Skill 覆盖以下 5 个汽车品牌的设计风格：

| 产品 | 风格描述 | 核心配色 | 字体 |
|------|---------|---------|------|
| BMW | Dark premium surfaces, precise German engineering aesthetic | Dark #1A1A1A, BMW Blue | BMW Type |
| Ferrari | Chiaroscuro black-white editorial, Ferrari Red with extreme sparseness | Black, White, Red #E8000D | Ferrari Font |
| Lamborghini | True black cathedral, gold accent, LamboType custom Neo-Grotesk | Pure Black, Gold #C9A84C | LamboType |
| Renault | Vivid aurora gradients, NouvelR proprietary typeface, zero-radius buttons | Aurora Gradient, White | NouvelR |
| Tesla | Radical subtraction, cinematic full-viewport photography, near-zero UI | White, Black | Gothic (Tesla专用) |

## 通用设计原则

汽车品牌的设计语言：极致视觉冲击，汽车摄影占据主视觉。极简 UI 哲学（尤其 Tesla）。奢侈品牌（Ferrari/Lambo）大量留白。科技品牌（Tesla/BMW）强调参数与性能。深色主题普遍，营造高端与力量感。

## 各产品详细设计规范

### Tesla
- **核心特征**：激进减法，电影级全视口摄影，接近零 UI
- **Hero**：车身大图从屏幕顶部到底部，白色文字叠加
- **按钮**：纯白/纯黑，无装饰，无阴影
- **字体**：Tesla Gothic，简洁无衬线
- **特点**：配置器、Autopilot、太阳能产品线

### Ferrari
- **核心特征**：明暗对比黑白编辑，Ferrari 红，极致稀疏
- **用色**：黑色、白色、极少量 Ferrari 红点缀
- **排版**：超大标题，大量负空间
- **图片**：赛车、跑车黑白大图，质感强
- **特点**：品牌 Heritage 网站、赛车内容

### Lamborghini
- **核心特征**：真正黑色大教堂，金色点缀，LamboType 定制无衬线
- **背景**：纯黑，强调犀利线条
- **字体**：LamboType，定制 Neo-Grotesk
- **金色**：仅用于高光点缀，极少使用
- **特点**：车型展示、定制配置器

### Renault
- **核心特征**：鲜明极光渐变，NouvelR 专属字体，零半径直角按钮
- **渐变**：Aurora 极光色，蓝绿紫渐变背景
- **字体**：NouvelR，专属法国设计
- **按钮**：零圆角，直角，激进几何感
- **特点**：电动车展示、创新科技

## 触发方式

用户说以下内容时激活：Tesla/Ferrari/Lamborghini 风格、汽车官网、汽车科技产品、高端品牌页。
