# Design & Creative Tools 设计风格 Skills

## 覆盖范围
本 Skill 覆盖以下 6 个设计/创意工具产品的设计风格：

| 产品 | 风格描述 | 核心配色 | 字体 |
|------|---------|---------|------|
| Airtable | Colorful, friendly, structured data aesthetic | Multi-color, Purple #24004D | Inter |
| Clay | Organic shapes, soft gradients, art-directed layout | Gradient Warm, Soft Purple | Inter |
| Figma | Vibrant multi-color, playful yet professional | Multi-color, Purple #A259FF | Inter |
| Framer | Bold black and blue, motion-first, design-forward | Black #000000, Blue #0045FF | Inter |
| Miro | Bright yellow accent, infinite canvas aesthetic | Yellow #FFD02F, White | Inter |
| Webflow | Blue-accented, polished marketing site aesthetic | Blue #146EF5, White | Inter |

## 通用设计原则

设计/创意工具产品的设计语言：

- **色彩丰富**：不惧怕多色混用，色彩是品牌语言的一部分
- **工具感强**：侧边工具栏、画布、属性面板是核心 UI 组件
- **协作友好**：实时协作指示、用户头像、评论气泡
- **动效丰富**：拖拽、过渡、动画预览是标配
- **空间开阔**：画布/编辑器区域最大化，UI 控件不占主视觉

## 各产品详细设计规范

### Figma
- **核心特征**：多彩活泼，专业且有趣，紫色品牌
- **侧边栏**：深色工具栏，图标按钮，激活态高亮
- **画布**：无限画布，白色背景，网格可选
- **属性面板**：右侧白色面板，细线分隔
- **特点**：组件系统、Auto Layout、设计令牌

### Framer
- **核心特征**：大胆黑白蓝，运动优先
- **按钮**：黑色填充，蓝色 hover，字体粗
- **画布**：深色/浅色切换，实时预览
- **特点**：网站构建器、动画编辑、组件属性

### Miro
- **核心特征**：明黄强调，无限画布感
- **工具栏**：顶部/左侧，明黄色图标
- **画布**：浅灰背景，支持无限拖拽
- **卡片**：便签式，明黄/粉/蓝多色 sticky notes
- **特点**：白板、思维导图、流程图

## 触发方式

用户说以下内容时激活：
- "用 Figma/Miro/Framer 风格"
- "做一个设计工具界面"
- "生成一个白板/协作画布"
- "做一个类似 Webflow 的可视化编辑器"

## 示例 Prompt

```
"帮我用 Figma 的多彩专业风格做一个设计工具界面"
"生成一个类似 Miro 的无限白板协作界面，明黄色点缀"
"做一个类似 Framer 的网站构建器界面，黑蓝大胆配色"
"帮我用 Airtable 的彩色友好风格做一个结构化数据管理工具"
```
