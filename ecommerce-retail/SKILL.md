# E-commerce & Retail 设计风格 Skills

## 覆盖范围
本 Skill 覆盖以下 4 个电商与零售产品的设计风格：

| 产品 | 风格描述 | 核心配色 | 字体 |
|------|---------|---------|------|
| Airbnb | Warm coral accent, photography-driven, rounded UI | Coral #FF5A5F, White | Nunito |
| Meta | Photography-first, binary light/dark surfaces, Meta Blue CTAs | Blue #0668E1, White | Inter |
| Nike | Monochrome UI, massive uppercase Futura, full-bleed photography | Black, White | Futura |
| Shopify | Dark-first cinematic, neon green accent, ultra-light display type | Dark #1A1A1A, Green #96BF48 | Inter |

## 通用设计原则

电商/零售产品的设计语言：摄影优先，产品图占据视觉中心。清晰的购买路径引导 Call-to-Action。品牌色在 CTA 按钮上高度统一。移动端购物体验优先。评分、评论、社会证明元素突出。

## 各产品详细设计规范

### Airbnb
- **核心特征**：温暖珊瑚色，摄影驱动，圆角 UI
- **按钮**：珊瑚色填充，白色文字，圆角大按钮
- **卡片**：白色背景，大圆角，阴影柔和，图片上方叠加文字
- **导航**：简洁顶部栏，搜索框居中，圆形用户头像
- **特点**：房源卡片、日历选择器、地图视图

### Shopify
- **核心特征**：深色电影院感，霓虹绿高亮，超浅显示字体
- **按钮**：霓虹绿填充，深色文字
- **Dashboard**：深色背景，霓虹绿图表线条
- **产品卡片**：深灰背景，大产品图，浅色文字
- **特点**：Admin Dashboard、商品管理、订单追踪

### Nike
- **核心特征**：单色 UI，超大 Futura 字体，全出血摄影
- **按钮**：黑/白极简按钮，无圆角，直角
- **字体**：Futura Condensed Extra Bold，超大尺寸
- **布局**：全出血 Hero 图，文字叠加在图片上
- **特点**：运动员摄影，产品 Detail 页面

## 触发方式

用户说以下内容时激活：Airbnb/Nike/Shopify 风格、电商网站、旅行平台、品牌官网。

## 示例 Prompt

```
"帮我用 Airbnb 的温暖珊瑚色风格做一个旅行预订平台首页"
"生成一个类似 Shopify 的深色电商管理 Dashboard"
"做一个类似 Nike 的运动品牌官网，超大字体全出血摄影"
```
